# 🌐 Portfolio Website

A premium, minimalist dark-themed developer portfolio built entirely from scratch using HTML, CSS, and Vanilla JavaScript. This project focuses on creating a modern, responsive, and interactive user experience while showcasing my frontend development skills and featured projects.

---

## ✨ Features

- 🌑 Modern dark-themed UI with clean typography.
- ⌨️ Custom typing animation built using Vanilla JavaScript.
- 📱 Fully responsive design for desktop, tablet, and mobile devices.
- 🎨 Smooth hover effects and micro-interactions.
- 🧭 Native smooth scrolling navigation.
- 💼 Dedicated project showcase section.
- ⚡ Built without any frontend frameworks or external JavaScript libraries.
